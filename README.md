# Developer Portfolio & Project Hub Dashboard

A premium, interactive developer dashboard built using HTML, CSS, and Vanilla JavaScript. Features real-time GitHub integration, an interactive Chart.js skills radar chart, a project progress tracker, and a mock contact inbox.

## Features

- **GitHub Stats Finder:** Enter any GitHub username to fetch profile details (avatar, bio, followers) and cards for the top repositories, showing stars, forks, and language.
- **Skills Radar Canvas:** Visualizes skills with an interactive Chart.js radar chart. Adjust skill sliders to watch the chart dynamically update in real-time.
- **Project Progress Board:** Add personal projects with custom status badges ("Planning", "In Progress", "Completed") and progress sliders. Tasks persist across page reloads using `localStorage`.
- **Contact Form Simulator:** Submit test messages that instantly display in the mock notifications/inbox panel.
- **Dual Themes:** Clean toggling between futuristic Dark/Neon and modern clean Light mode.

## How to Run

1. Open `index.html` directly in your favorite web browser (double-click the file).
2. Alternatively, run a local web server (e.g., using VS Code Live Server extension or Python `python -m http.server`) for the best local development experience.
